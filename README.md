# PoliceDispatchLua
Скрипт с звуковой озвучкой полицейского диспетчера в GTA:SAMP. Доступны настройки для следующих SAMP серверов:
- Arizona RP
- Evolve RP
- Trinity RP
- Diamond RP
- Gambit RP
- Training
- Revent RP

Дальнейшая разработка продолжается.

### Обзор скрипта с диспетчером на канале Brothers In Company:
