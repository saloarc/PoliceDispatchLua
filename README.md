# PoliceDispatchLua
Звуковой движок с озвучкой полицейского диспетчера в GTA:SAMP. Присутствуют темы на форумах проектов: 
- Blasthk,
- Diamond,
- Gambit.

По отзывам игроков, не работает на проекте Diamond. Дальнейшая разработка отменена.
